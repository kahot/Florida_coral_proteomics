
# Florida coral proteomics daya analysis

## This repository contains the analysis scripts and associated data for Florida corals (*Orbicella faveolata* and *Acropora cervicornis*) proteomics.

The following Skyline output files (main input files for the analysis) are not available here due to the file sizes:

	- MSstats_format_acerv.csv
	- MSstats_ofav.csv


- For the analysis of *O.favelota* proteomic data, see  DIA_LIMMA_Ofav.Rmd
- For the analysis of *A.cervicornis* proteomic data, see  DIA_LIMMA_Acerv.Rmd
